##Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos de Python:

Estruturas de repetição e decisão (while, if, elif, else)

Manipulação de arquivos de texto (open, read, write, close)

Tratamento de erros e exceções com try e except

Criação e reutilização de funções

Construção de um menu interativo em console

##O que aprendi

Durante o desenvolvimento, consegui compreender:

Como validar entradas do usuário para evitar erros.

Como salvar dados localmente de forma simples.

Como dividir o código em funções para facilitar a leitura e manutenção.

A importância de tratar exceções para evitar falhas inesperadas.
